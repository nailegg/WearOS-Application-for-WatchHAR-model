# Notice

This repository is a portfolio-oriented extraction of a Wear OS human activity
recognition application implemented by Sanghyeon Lee during an internship
project.

The repository intentionally excludes raw datasets, large experiment runs,
training checkpoints, IDE files, and unrelated collaborative research history.
The included files focus on the Android application runtime and compact
supporting documentation.

Bundled TFLite model assets are included to keep the application runnable in a
local snapshot. Their redistribution status should be confirmed before public
publication.

The SAMoSA dataset distribution report summarizes a local audit. Raw data is not
included in this repository.
